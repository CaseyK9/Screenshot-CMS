# Screenshot CMS
Screenshot CMS is a simple web application to manage your taken screenshots.

## How to install?
Please take a look at our <a href="https://github.com/siemen6/Screenshot-CMS/wiki/Installation">wiki</a>.

## Features
- Import screenshots
- Access log
- Private upload key
- Advanced settings
- Bootstrap 4 Design

## Upcoming features
- Dark theme
- Backup function
- Statistics
